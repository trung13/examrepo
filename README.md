
1. Cài đặt môi trường

    ```bash
    sudo apt-get update 
    sudo apt-get install -y vim python3-pip curl git
    pip3 install --upgrade pip
    pip install docker-compose
    ```

2. Cài Docker 

   ```bash
   sudo curl -sSL get.docker.com | sh
   ```


3. Clone repo

    ```bash
    git clone 
    ```

4. Khởi động

    ```bash
    docker-compose up -d
    ```
